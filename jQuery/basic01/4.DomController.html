jQuery DOM 조작 메서드 예시

여기서 문단을 추가하거나 변경할 수 있습니다.

부모 요소의 자식 요소

이 요소는 여러 메서드에서 사용됩니다.

.append(): 선택한 요소의 끝에 새 HTML 콘텐츠를 추가합니다. 예를 들어, #container의 끝에 새 문단을 추가합니다.
.prepend(): 선택한 요소의 시작 부분에 새 HTML 콘텐츠를 추가합니다. 예를 들어, #container의 처음에 새 문단을 추가합니다.
.before(): 선택한 요소 앞에 새 HTML 콘텐츠를 추가합니다. 예를 들어, #target 요소의 앞에 새 문단을 추가합니다.
.after(): 선택한 요소 뒤에 새 HTML 콘텐츠를 추가합니다. 예를 들어, #target 요소의 뒤에 새 문단을 추가합니다.
.html(): 선택한 요소의 HTML 콘텐츠를 변경합니다. 예를 들어, #container의 내용을 새로운 HTML로 변경합니다.
.text(): 선택한 요소의 텍스트 콘텐츠를 변경합니다. 예를 들어, #container의 텍스트를 새 텍스트로 변경합니다.
.remove(): 선택한 요소를 DOM에서 제거합니다. 예를 들어, #target 요소를 삭제합니다.
.empty(): 선택한 요소의 자식 요소를 모두 제거합니다. 예를 들어, #container 내의 모든 자식 요소를 삭제합니다.
.replaceWith(): 선택한 요소를 새로운 콘텐츠로 교체합니다. 예를 들어, #target 요소를 새 HTML로 교체합니다.
.clone(): 선택한 요소를 복제하여 반환합니다. 예를 들어, #target을 복제하여 #container에 추가합니다.
.attr(): 요소의 속성 값을 가져오거나 설정합니다. 예를 들어, #target의 클래스를 변경합니다.
.removeAttr(): 선택한 요소에서 특정 속성을 제거합니다. 예를 들어, #target에서 클래스를 제거합니다.
.css(): 선택한 요소의 스타일 속성을 가져오거나 설정합니다. 예를 들어, #target의 텍스트 색을 빨간색으로 설정합니다.
.addClass(): 선택한 요소에 클래스를 추가합니다. 예를 들어, #target에 새 클래스를 추가합니다.
.removeClass(): 선택한 요소에서 클래스를 제거합니다. 예를 들어, #target에서 기존 클래스를 제거합니다.
.toggleClass(): 선택한 요소에 클래스를 토글합니다. 예를 들어, #target에 클래스가 있으면 제거하고 없으면 추가합니다.
.val(): 선택한 입력 요소의 값을 가져오거나 설정합니다. 예를 들어, #input 요소의 값을 변경합니다.
.prop(): 선택한 요소의 속성 값을 가져오거나 설정합니다. 예를 들어, #checkbox 선택 상태를 변경합니다.
