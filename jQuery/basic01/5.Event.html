jQuery 이벤트 처리 메서드 예시

마우스 들어감 - .mouseenter()
마우스 나감 - .mouseleave()
마우스 이동 - .mousemove()












.click(): 버튼 클릭 시 알림을 보여주는 이벤트 처리 메서드입니다.
.dblclick(): 더블클릭 시 알림을 보여주는 이벤트 처리 메서드입니다.
.mouseenter(): 마우스가 요소 위로 올라가면 알림을 보여주는 이벤트 처리 메서드입니다.
.mouseleave(): 마우스가 요소에서 나가면 알림을 보여주는 이벤트 처리 메서드입니다.
.mousemove(): 마우스가 요소 위에서 이동할 때 알림을 보여주는 이벤트 처리 메서드입니다.
.keydown(): 키를 눌렀을 때 발생하는 이벤트 처리 메서드입니다. 예를 들어, 텍스트 박스에 키를 누르면 알림을 보여줍니다.
.keypress(): 키가 눌릴 때 발생하는 이벤트 처리 메서드입니다. 특히 문자 입력에 유용합니다.
.keyup(): 키를 떼었을 때 발생하는 이벤트 처리 메서드입니다.
.focus(): 입력 필드에 포커스가 올 때 발생하는 이벤트 처리 메서드입니다. 예를 들어, 텍스트 필드에 클릭하면 알림을 보여줍니다.
.blur(): 입력 필드에서 포커스를 잃을 때 발생하는 이벤트 처리 메서드입니다.
.change(): 셀렉트 박스에서 선택된 옵션이 변경될 때 발생하는 이벤트 처리 메서드입니다.
.submit(): 폼이 제출될 때 발생하는 이벤트 처리 메서드입니다.
.on(): 여러 이벤트를 처리할 때 유용한 메서드입니다. 예를 들어, 여러 이벤트를 한번에 처리할 수 있습니다.
.off(): 지정한 이벤트 리스너를 제거하는 메서드입니다.
.trigger(): 지정한 이벤트를 강제로 발생시키는 메서드입니다.
